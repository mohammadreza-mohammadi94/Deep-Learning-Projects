The model uses a univariate target (Temperature) and multivariate input features across a rolling time window to make predictions.

* **Architecture:**

* **Sequence Length:** 24 hours (1 day of hourly data)
* **Loss:** Mean Squared Error (MSE)
* **Optimizer:** Adam
